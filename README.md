# HydroFlash V1

Digital Twin dashboard for the HydroNet municipal water system.

![CI Status](https://github.com/jonhouck/hydroflashv1/actions/workflows/ci.yml/badge.svg)

## Features
- **Network Visualization**: Interactive pipe network using React Flow.
- **Real-time Telemetry**: Simulated pressure and flow data.
- **Custom Assets**: Reservoirs, Pumps, Tanks, Junctions.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser.

## Testing
Run unit and integration tests:
```bash
npm test
```
